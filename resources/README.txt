The resources directory is the base directory for various resources needed:

colormaps directory
===================

Images that are describing terrain colors per vertex in the terrain grid.

fonts directory
===============

Fonts as produced by the Hiero (https://github.com/libgdx/libgdx/wiki/Hiero)
font generator. Both the font files and the font textures are in this directory.

heightmaps directory
====================

Images (ordinary bitmaps or RAW format) that are describing terrain heights
per vertex in the terrain grid.

shaders directory
=================

Shader programs used by piece-of-earth.

texture directory
=================

Textures used by piece-of-earth.

models directory
================

Wavefront model files (.obj).
